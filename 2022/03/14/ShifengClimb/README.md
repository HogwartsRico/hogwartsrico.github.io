# 缘起

2022年3月12日，杭州进入了早春三月，天气宜人，遂决定去爬龙井产地，狮峰山。 

# 路线

打车到法喜寺，从法喜寺对面的入口上山，爬到狮峰。然后下山便到了龙井村。 总长约2公里。 

![](轨迹.png) 

然后再从龙井村骑车绕西湖半圈，约10KM

每年的三四月份，是杭州最美的季节：西湖中水光潋滟，狮峰山新茶吐芽，断桥上游人如织。

在三月的杭州走一走，真是一番享受！
